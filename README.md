# MEPIUS Project Notice

메피우스 프로젝트는 의료진단 플랫폼을 구축하고 의료산업에 공헌하는 프로젝트로 건강 데이터를 블록체인에 보관하고 이용자와 파트너가 균등하게 혜택받는 안전한 디지털 헬스케어 생태계를 만듭니다.

## Project Goal
이 프로젝트는 메피우스 프로젝트의 진행 사항을 공유하기 위한 내용을 관리하는 프로젝트입니다.
